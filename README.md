## 📝 Descrição do Projeto
Este repositório contém a resolução completa das 11 questões propostas para a disciplina. O trabalho une a análise de requisitos, o projeto de diagramas de classe UML e a implementação prática das interfaces.

## 📂 Estrutura de Arquivos
* **Diagramas**: Imagem contendo os 11 diagramas de classe (Atributos e Métodos).
* **Requisitos**: Arquivo `requisitos.txt` com a descrição de cada sistema.
* **Códigos**: Arquivos de `questao01.py` a `questao11.py` desenvolvidos em Python/Streamlit.

## 🛠️ Tecnologias
- **UML**: Para modelagem.
- **Python/Streamlit**: Para o desenvolvimento do software.
- **GitHub**: Hospedagem oficial na branch master.
